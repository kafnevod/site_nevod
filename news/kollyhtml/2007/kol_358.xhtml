ТРУДОВЫЕ СЕПУЛЬКИ

В числе забавных документов, сопровождающих нашего экс-советского человека по жизни, есть ТРУДОВАЯ КНИЖКА. Последние годы государство над ней дополнительно поработало. Создало новый ее вариант, который действует наряду со старым. Ввело ряд изменений в правила ведения трудовой книжки. Проделана большая тонкая работа, которая понятна только очень узким специалистам, а для остальных лишь добавила путаницы. Но нет ответа на вопрос: "Кому и зачем нужна сегодня трудовая книжка?"

Сегодня, при переходе на персонифицированный учет, отмене непрерывного стажа и тому подобных вещей, трудовая книжка абсолютно не нужна для начисления пенсий и всего прочего, для чего она раньше использовалась. Разве что для записи благодарностей.

Нужна сегодня трудовая книжка только для создания дополнительных рабочих мест в отделах кадров предприятий и организаций.

У фантаста Станислава Лема есть в его "межгалактическом толковом словаре" слово сепулька. В качестве объяснения этого странного слова написано, смотри сепуление. В качестве объяснения слова сепуление написано, смотри сепуляция. В качестве объяснения слова сепуляция написано, смотри сепульки. КРУГ ЗАМКНУЛСЯ! Это пример слова, которое существует лишь само для себя, поскольку не имеет семантики (смысла).

Так вот, сегодня трудовые книжки существуют сами для себя (и для подкормки бюрократов) их заполнение потеряло всякий смысл, поскольку их данные нигде не используются. Зато, сколько с ними маяты.
